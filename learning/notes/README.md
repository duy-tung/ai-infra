# Notes — ghi chú học theo phase

Ghi note ngắn, có cấu trúc, **gắn với artifact**. Mục tiêu của note không phải
chép lại lesson, mà là: (1) chốt insight, (2) link tới code bạn build, (3) ghi
câu hỏi mở.

## Cách dùng

- Mỗi phase 1 file (`phase-XX-*.md`). Dùng [`_template.md`](_template.md) cho mỗi lesson.
- Note bằng **tiếng Việt cho dễ nghĩ**, nhưng phần "insight" và "1-line summary"
  cố viết **tiếng Anh** (tái sử dụng cho blog/README/CV).
- Mỗi note nên trả lời: *"điều này giúp ích gì cho project nào trong `projects/`?"*
