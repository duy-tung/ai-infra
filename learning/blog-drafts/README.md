## Gợi ý publish

- Đăng dạng series trên dev.to / Medium / blog cá nhân, hoặc gộp vào README repo.
- Mỗi bài kèm: architecture diagram, 1 GIF demo, link repo.
- Bài flagship dùng làm "pinned" / mục About trong CV/LinkedIn.
- Mỗi repo: điền section "What I learned / Failure modes" (đã có draft trong README).
